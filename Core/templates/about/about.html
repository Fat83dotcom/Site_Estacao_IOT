Sobre o projeto:

A estação meteorológica Brainstorm Tecnologia é um projeto experimental destinado à aplicação prática de conceitos e tecnologias adquiridos durante o curso de Análise e Desenvolvimento de Sistemas. Seu objetivo principal é alimentar um banco de dados com informações sobre temperatura, umidade e pressão atmosférica, coletadas por sensores distribuídos em vários locais, os quais são integrados automaticamente ao sistema através do firmware desenvolvido especificamente para o projeto.

A coleta de dados é realizada utilizando um ESP32 em conjunto com um sensor BME280, ambos conectados a uma rede doméstica Wi-Fi.

O controle dos sensores é realizado por um cliente MQTT, que, por meio de um sistema de publicação/assinatura, envia solicitações aos sensores, os quais respondem com os dados correspondentes. Esses dados são então processados e armazenados no banco de dados pelo cliente MQTT.

Por fim, um aplicativo web foi desenvolvido para possibilitar a visualização dos dados gravados. Para acessar essas informações, basta selecionar o sensor desejado na lista de sensores disponíveis.

Para obter mais detalhes técnicos e acesso ao código-fonte do projeto, visite o portfólio do desenvolvedor .

Se você deseja contribuir com o projeto fornecendo um sensor, entre em contato através do e-mail fernando.quimic@gmail.com.
